Step 3 — How to Place on Each Website
Format A — Text / Button Link
Simplest. Replace the href on any existing button or text link.
<!-- Paste on betmgmbonuscode.us --> <a href="https://track.apogeemobi.com/track/click/CAMP_TOKEN ?pid=PID_USA_BET &sub1=betmgmbonuscode &clickid=VISITOR_ID" target="_blank" rel="noopener"> Get BetMGM $1000 Bonus → </a>
✓ Works on WordPress, Webflow, plain HTML, any CMS
Format B — Banner Image Link
Wrap your banner image in the tracking link.
<!-- Banner ad on carinsuranceguide.us --> <a href="https://track.apogeemobi.com/track/click/CAMP_TOKEN ?pid=PID_USA_INS &sub1=carinsurance &clickid=VISITOR_ID" target="_blank"> <img src="banner-geico-728x90.jpg" alt="Get Car Insurance Quote" style="width:100%;max-width:728px" /> </a>
Format C — JavaScript (auto unique clickid)
Generates a unique visitor ID automatically. Recommended for accurate attribution.
<!-- Place in <head> once per page --> <script> // Generate unique visitor ID once per session window.APOGEE_VID = sessionStorage.getItem('apg_vid') || (sessionStorage.setItem('apg_vid', Date.now()+'-'+Math.random().toString(36).slice(2,8)), sessionStorage.getItem('apg_vid')); // Helper to build tracking URL function trackURL(token, pid, sub1) { return 'https://track.apogeemobi.com/track/click/' + token + '?pid=' + pid + '&sub1=' + sub1 + '&clickid=' + window.APOGEE_VID; } </script> <!-- Then on any button --> <a id="offer-btn" href="#" onclick="this.href=trackURL( 'CAMP_TOKEN', 'PID_USA_BET', 'betmgmbonuscode')"> Claim Bonus </a>

For WordPress sites — use a redirect plugin (Pretty Links, ThirstyAffiliates) or just paste the link directly.

Step 4 — The clickid= Parameter (Critical for Attribution)
What is clickid= ?

A unique ID your website generates per visitor. It ties the click to the conversion. Without it, if 100 users click but only 10 convert, you won't know which 10 to credit.

✓ Must be unique per visitor
✓ Max 64 characters
✓ No spaces or special chars
✓ Store it so you can look it up later
Options — pick one
// Option 1 — Timestamp + random (simplest) const cid = Date.now() + '-' + Math.random().toString(36).slice(2,8); // e.g. 1778141817-k9mx2p // Option 2 — Session storage (same ID whole visit) const cid = sessionStorage.getItem('cid') || crypto.randomUUID().replace(/-/g,'').slice(0,16); sessionStorage.setItem('cid', cid); // Option 3 — WordPress user cookie const cid = getCookie('wordpress_logged_in') || uuid();
