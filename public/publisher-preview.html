Each publisher auto-gets a unique pid= token

When a publisher group is created, the platform auto-generates a short token. This token goes in the pid= parameter of every tracking link from that group.

You also add notes listing which specific websites belong to this group. The individual site is tracked via sub1=.

✓ One publisher = one group of related sites
✓ sub1= tracks which individual site sent the click
✓ No login needed for your websites — just a link

What happens when user clicks (in <50ms)
1. Our platform receives the click
2. Records: pid=jm7dXaWAap (USA Betting)
3. Records: sub1=betmgmbonuscode (exact site)
4. Records: device, geo, OS, time
5. Generates unique click_id=Xk9mPqR2wT
6. 302 redirect → BetMGM offer page
✓ User sees no delay — redirect is instant
✓ Click is recorded even if user doesn't convert
